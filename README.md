## ✨ ویژگی‌ها

- ⚡ بررسی لحظه‌ای با متد `HEAD`
- 🎨 تم دارک حرفه‌ای
- 🔄 بروزرسانی خودکار هر ۳۰ ثانیه
- ⏱️ نمایش زمان پاسخ به میلی‌ثانیه
- 📱 کاملاً واکنش‌گرا
- 🎯 مدیریت سایت‌ها با فایل JSON

## 🚀 اجرا

```bash
git clone https://github.com/[username]/status-page.git
cd status-page
```

فایل `sites.json` رو ویرایش کن:

```json
[
    { "name": "گوگل", "url": "https://www.google.com" },
    { "name": "گیت‌هاب", "url": "https://github.com" }
]
```
